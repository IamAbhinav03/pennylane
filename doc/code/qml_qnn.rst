qml.qnn
=======

.. currentmodule:: pennylane.qnn

This module contains functionality for converting PennyLane QNodes into layers that are compatible
with Keras and PyTorch.

.. note::

    Check out our `Keras <https://pennylane.ai/qml/demos/tutorial_qnn_module_tf.html>`__
    and `Torch <https://pennylane.ai/qml/demos/tutorial_qnn_module_torch.html>`__ tutorials for
    further details.
